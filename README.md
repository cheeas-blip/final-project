# Final Project

## Overview
This project is a full-stack dashboard using an Express.js backend and a frontend interface that fetches live API data.

## Features
- GET /api/items route
- POST /api/items route
- Express backend with middleware
- Environment variable integration
- Live deployment
- Loading and error handling UI

## Deployment Steps
1. Created Express server (server.js)
2. Built frontend (HTML, JS, CSS)
3. Connected frontend to backend using fetch
4. Uploaded project to GitHub
5. Deployed using Render
6. Added environment variable in Render
